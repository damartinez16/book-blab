# BOOK BLAB 
Book Blab: A place to blab about books. 
Add your favorite books, review other books, and comment how much you loved them. A perfect book place for book people.

- HTML, CSS, and JavaScript
- Express
- Node.js
- Mongo DB & Atlas
- Mongoose
- Google OAuth
- Heroku
